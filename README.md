# 📊 Customer Churn Prediction

## 📌 Project Overview

This project develops a Machine Learning model to predict customer churn in a telecommunications company.

The objective is to identify customers at risk of cancellation and support customer retention strategies.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## 🔍 Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Encoding
5. Model Training
6. Model Evaluation

---

## 🤖 Models Evaluated

| Model | Accuracy | AUC-ROC |
|---------|---------|---------|
| Logistic Regression | 79.5% | 0.836 |
| Decision Tree | 81.5% | 0.867 |
| Random Forest | 82.3% | 0.872 |

---

## 📈 Key Findings

- Month-to-month contracts show the highest churn rates.
- Fiber optic customers exhibit higher cancellation levels.
- Lower customer tenure is strongly associated with churn.
- Higher monthly charges increase churn risk.

---

## 🏆 Final Model

Random Forest achieved the best performance:

- Accuracy: 82.3%
- AUC-ROC: 0.872

---

## 💼 Business Recommendations

- Promote long-term contracts.
- Implement early retention strategies for new customers.
- Investigate dissatisfaction factors among fiber optic users.
- Offer personalized plans to high-risk customers.
